## -*- coding: utf-8 -*- <%inherit file="/base.mako"/> <%namespace file="/extensions.mako" import="*"/> ##name Data Embedding You can embed data into the `settings.py` file. This python file is imported automatically into the namespace. (If you change `settings.py` you will need to restart the pyblue server) Using the current `settings.py` file we can write the following: <%text filter="n"> % for author, title, url, journal in p.settings.publications(): ${author} ${title} ${journal} % endfor To produce the output % for author, title, url, journal in p.settings.publications(): % endfor
Authors Title Journal
${author} ${title} ${journal}
