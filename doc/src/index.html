Welcome to Pyblue!

Start with a plain index.html file in a directory.

  1. Launch pyblue serve -f doc/src
  2. View your site by visiting http:://localhost:8080.
The pages are rendered through the Django Template engine. Users get additional context for each of the pages.
  • Writing: {% verbatim %} {{ p.last_modified }} {% endverbatim %} will produce {{ p.last_modified }}
There are many other additional context variables, some can be specified in the file itself..

To access more advanced features add the pyblue tag loader to the page. {% verbatim %} {% load pytags %} {% endverbatim %}

{% load pytags %} You can now make use of specific tags for example: {% verbatim %} {% link "context" %} {% endverbatim %} will produce the following: {% link "context" %}.

Why? The {% verbatim %} {% link "context" %} {% endverbatim %} command performs a regular expression search on all files in the directory and once it finds a match it produces a link to it with the proper path. This keeps links correct even after moving files!
