Navigate

To navigate link to files with link(pattern). The pattern will be interpreted as a regular expression.:

  • link('index') will produce Welcome to Pyblue
  • If multiple files match the first will be picked: link('b') will produce About
  • If the pattern does not match you'll get a special display link('joe') will produce Link pattern 'joe' does not match!
